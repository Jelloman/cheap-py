# cheap-core

Core Cheap interfaces and basic implementations (minimal dependencies).

This package provides the foundation of the Cheap data caching system, implementing the CHEAP model:
- **C**atalog: Container for related data
- **H**ierarchy: Organizational structure for entities
- **E**ntity: Individual data items with unique identifiers
- **A**spect: Key-value properties associated with entities
- **P**roperty: Individual data values within aspects

## Installation

```bash
pip install cheap-core
```
